&&& ## Descrption du projet:
 Il s'agit d'une application d'e-commerce dédiée à la vente des gadgets électroniques.L'utilisateur peut sur la page d'accueil consulter les produits disponibles sur l'application, et aussi avoir les détails de chaque produit, il a également la possibilité de gérer un panier et ajoutant et supprimant des paroduits. L'application intègre un système d'authentification et meme un processus de rénitialisation de mot de passe. 
Plus de détails sur le cahier de charge:https://docs.google.com/document/d/1EeVjDSLJZvaAkE5EV9211w89adsZJN82H6yXp9gfk3I/edit?usp=sharing
Et la maquette figma : https://www.figma.com/design/bwpXUNnZCLUuKq4PRkLJj2/e-tech?node-id=28-781&t=fQ1b5b03pjJz1LFF-1

## Compilation du projet :
Pour lancer l'api : 
    - cd api_mobile/api
    - venv/Scripts/activate
    - python manage.py runserver 
    le serveur devrait se lancer au http://127.0.0.1:8000/

Pour lancer l'application flutter :
    - cd front 
    - flutter run -d chrome
